Integrated Loudness
--LUFS
楽曲全体の平均的な音の大きさ。配信では -14〜-7 LUFS が一般的。
True Peak
--dBTP
瞬間的な音量の最大値。0 dBTP を超えると音割れの原因になる。-1 dBTP 以下が安全。
Loudness Range
--LU
曲中の音量差の幅。値が大きいほどダイナミクスが広い。
Head / Tail Silence
--s / --s
冒頭・末尾の無音区間。納品時に長すぎないか確認。
Duration
--
楽曲の長さ。
Zero Start / End
-- / --
先頭・末尾のサンプル値が 0 か。非ゼロだとプチッとノイズが出る。
Clipping
--samples
0 dBFS に張り付いたサンプル数。0 が理想。
Stereo Correlation
--
L/R の相関。1.0 = モノラル、0 付近 = 広いステレオ、負 = 位相問題あり。
